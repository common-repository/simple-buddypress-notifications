== Description ==

Simple Buddypress Notifications allows sites using BuddyPress to have a notifications icon on the site. It also allows a ting sound on the notifications.
Use [simple_buddypress_notifications] to show notifications icon.
The icons shows live notifications as well as offline ones.

Notification audio attribution to https://github.com/akx/Notifications

== Installation ==

This section describe how to install the plugin and get it working.


    Upload Simple Buddypress Notifications to the /wp-content/plugins/ directory

    Activate the plugin through the ‘Plugins’ menu in WordPress
